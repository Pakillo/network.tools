# Bipartite network

Example data frame containing data for a bipartite web of number of
visits from 8 animals to 4 plants.

## Usage

``` r
web
```

## Format

### `web`

A data frame with 32 rows and 3 columns:

- Plant:

  Plant ID

- Animal:

  Animal ID

- Visits:

  Number of visits
