# Changelog

## network.tools 0.0.5

- `indiv_spec` now returns a dataframe with WIC, TNW and WIC/TNW
  columns. To obtain individual Shannon diversity values, use
  `WIC(net, indiv = TRUE)`.

## network.tools 0.0.4

- Add functions to calculate niche width and individual specialisation
  indices.

## network.tools 0.0.3

- Horizontal centering of animal names in plot_web_heatmap.

## network.tools 0.0.2

- Add functions to remove empty (all zero) rows and columns from a
  matrix, following [@elequintero](https://github.com/elequintero)
  suggestion.
